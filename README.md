# Kethleen Bruno — Portfolio

Site pessoal de Kethleen Bruno, UX Researcher & Product Designer.

🔗 **Publicado em:** https://kethbruno.github.io/newportfolio/

## Sobre

Portfólio apresentando projetos, experiência profissional, capacidades e depoimentos, com foco em pesquisa de UX aplicada a operações de serviço e transformação digital.

## Estrutura

```
index.html      → página única (HTML, CSS e JS puros, sem build/framework)
assets/         → imagens, logos e animações usadas no site
```

## Tecnologia

- HTML, CSS e JavaScript puros — sem dependências externas além das fontes do Google Fonts.
- Hospedado via GitHub Pages.

## Atualizando o site

1. Edite `index.html` diretamente.
2. Para trocar/adicionar logos do carrossel de clientes, atualize o array `logos` no `<script>` ao final do arquivo, apontando para um arquivo correspondente em `assets/`.
3. Após alterar, faça commit e push para a branch `main` — o GitHub Pages atualiza automaticamente em 1–2 minutos.
